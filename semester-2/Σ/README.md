# Автозадание - Σ
Напишите адрес клиентского приложения, которое нативным образом обрабатывает адрес вида:

https://abc.def/?A=2,4,6,8,1&B=1,-1,1,-1,1&C=4

Где:

A — ряд целых чисел (вектор входов) длины n

B — ряд целых чисел (вектор весов) длины n

C — целое число (смещение, bias)

Нужно вычислить:

z = Σ(A[i] * B[i]) + C

y = σ(z) = 1 / (1 + e^(−z))

и поместить результат в заголовок окна (document.title) в формате:

0.xxx — три знака после точки без округления (простое отсечение),

либо 1

Дополнительно:

В заголовке h4 должен быть ваш логин в этой системе (без пробелов и переводов строк).

Пожалуйста, напишите адрес до знака вопроса

НЕ используйте в коде console, а также свойство searchParams.
(Можно использовать location.search, а также строковые/массивные методы JS.)

Просьба избегать имён переменных, совпадающих с встроенными API браузера (например URL).
